# 📚 LitHub

**LitHub** is a native Android e-Library application developed in Java that allows users to read and download books seamlessly. The app is powered by Firebase for backend services and provides an admin panel for managing digital content.

---

## 📱 Overview

LitHub is designed as a digital library platform where:

- Users can browse, read, and download books  
- Admins can upload and manage book content  
- Firebase handles storage, authentication, and database operations  

---

## 🚀 Features

### 👤 User Features
- 📖 Read books online  
- ⬇️ Download books for offline access  
- 🔐 User authentication (Sign up / Login)  
- 🔍 Browse available books  

### 🛠️ Admin Features
- ➕ Upload new books  
- ❌ Delete existing books  
- 📂 Manage library content efficiently  

---

## 🛠️ Tech Stack

- **Language:** Java  
- **Platform:** Native Android  
- **Backend:** Firebase  
  - Firebase Authentication  
  - Firebase Firestore  
  - Firebase Storage
